# CWA 資料集

Phase 1 實測後，API 使用下列 CWA Open Data 資料集。

| 用途 | Dataset ID | 狀態 | 說明 |
|------|------------|------|------|
| 即時觀測 | `O-A0003-001` | 可用 | 自動氣象站觀測資料，用於最近測站、氣溫、濕度、雨量、風速等 |
| 鄉鎮 3 小時預報 | `F-D0047-001`～`F-D0047-087` 奇數分檔 | 可用 | 依最近測站縣市選擇縣市分檔，例如臺北市使用 `F-D0047-061` |
| 全台縣市 fallback | `F-D0047-089` | 可用 | 縣市分檔或鄉鎮匹配失敗時的 fallback |
| 全台鄉鎮資料 | `F-D0047-093` | 目前 API 404 | 公開文件仍列此資料集，但本機 API 實測為 `Resource not found` |

### O-A0003-001 要素可用性（實測概況）

| 要素 | 說明 |
| --- | --- |
| 氣溫、濕度、風速、風向、10 分鐘雨量 | 多數測站有值 |
| 氣壓、UV | 僅少數測站提供；其餘 CWA 回傳 `-99`（缺值），API 會轉為 `null` 並顯示「無資料」 |
| 風向 | 約九成測站有值；最近測站若缺值同樣顯示「無資料」 |

最近測站依距離選取，**不會**為了補齊氣壓或 UV 改選其他測站。
